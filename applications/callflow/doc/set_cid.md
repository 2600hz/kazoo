## Set Cid

### About Set Cid

#### Schema

Validator for the set_cid callflow data object



Key | Description | Type | Default | Required | Support Level
--- | ----------- | ---- | ------- | -------- | -------------
`caller_id_name` | Set caller ID Name | `string()` | "" | `false` |  
`caller_id_number` | Set caller ID number | `string()` | "" | `false` |
