# Transactions

## Get transactions

### Request

- Verb: `GET`
- Url: `/accounts/{{ACCOUNT_ID}}/transactions`
- Payload: None
- options:
    - `created_from` {{TIMESTAMP}}
    - `created_to` {{TIMESTAMP}}
    - `reason` `only_calls`

## Get current balance

### Request

- Verb: `GET`
- Url: `/accounts/{{ACCOUNT_ID}}/transactions/current_balance`
- Payload: None

### Response

    {"data": {
        "balance": 9.18
     },
     "status": "success"
    }
