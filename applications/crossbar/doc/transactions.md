#### Debit an account

Only for super duper admins and resellers.

> DELETE /v2/accounts/{ACCOUNT_ID}/transactions/debit

```curl
curl -x DELETE \
     -H "X-Auth-Token: {AUTH_TOKEN}" \
     -d '{"data": {"amount": 1} }'
     http://{SERVER}/v2/accounts/{ACCOUNT_ID}/transactions/debit
{
    "auth_token": "{AUTH_TOKEN}",
    "data": {
        "metadata": {
```
