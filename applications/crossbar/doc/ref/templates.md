#### Fetch

> GET /v2/accounts/{ACCOUNT_ID}/templates

```shell
curl -v -X GET \
    -H "X-Auth-Token: {AUTH_TOKEN}" \
    http://{SERVER}:8000/v2/accounts/{ACCOUNT_ID}/templates
```
