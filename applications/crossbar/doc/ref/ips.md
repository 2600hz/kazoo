# Ips

## About Ips

#### Schema

IP addresses assigned to the account



Key | Description | Type | Default | Required | Support Level
--- | ----------- | ---- | ------- | -------- | -------------
`ips.[]` |   | `string()|string()` |   |   |  
`ips` | List of IP addresses | `array()` |   | `false` |  



## Fetch

> GET /v2/accounts/{ACCOUNT_ID}/ips
