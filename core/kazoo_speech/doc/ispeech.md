# iSpeech

## Configuration

```
       "tts_api_key": "",
       "tts_frequency": "16000",
       "tts_bitrate": "16",
       "tts_speed": 0,
       "tts_start_padding": 1,
       "tts_end_padding": 0,
       "tts_provider": "ispeech",
       "tts_voice": "female",
       "tts_language": "en-US",
       "tts_default_voice": "female/en-US",
       "tts_url_ispeech": "",
       "tts_cache": 3600000,
       "default_provider": "ispeech"
```
